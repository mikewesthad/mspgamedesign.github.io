--- layout: page title: Platformer Game --- Platformer games were built using the Stencyl game engine. Students first designed storyboards and shared their game narrative to the class to receive feedback. Once their ideas were complete, students then began building the game on the computer using Photoshop to design characters and scenes before exporting them into Stencyl. Components such as physics and other character properties were designated in Stencyl, which encourages the use of logic similar to that of computer programming.
